Privacy Policy

Last updated: January 2025

1. Information We Collect

A-OS Kernel collects minimal data necessary for system operation:

  • Operator authentication credentials (username, hashed passwords)
  • System event logs and kernel metrics
  • Mesh node synchronization data
  • Agricultural and transport domain data (when applicable)

2. How We Use Your Data

Your data is used exclusively for:

  • System authentication and authorization
  • Event monitoring and system health tracking
  • Cross-village data synchronization
  • Domain-specific operations (agriculture, transport, health, religion)

3. Data Storage

All data is stored locally in SQLite databases on your sovereign node. No data is transmitted to external servers unless explicitly configured for mesh synchronization.

4. Data Sharing

A-OS operates on a sovereign architecture. Your data remains on your node and is only shared with explicitly configured mesh peers for synchronization purposes.

5. Your Rights

As a sovereign system operator, you have full control over your data:

  • Access all stored data via the dashboard
  • Export data in standard formats
  • Delete data at any time
  • Control mesh synchronization settings

6. Security

A-OS implements enterprise-grade security measures including JWT authentication, encrypted mesh communication, and role-based access control (RBAC).

7. Contact

For privacy-related inquiries, contact your system administrator or the A-OS development team.
